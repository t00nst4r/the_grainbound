About This Project

How a software engineer with ADD accidentally systematized worldbuilding.

The Problem

I've been worldbuilding for about ten years—ever since I discovered D&D. I've built multiple worlds as a DM and tried my hand at writing, but my severe perfectionism and ADD kept me from finishing anything.

"I kept getting stuck at the same point. Every. Single. Time. It wasn't that I wasn't creative—it was that I'd lost oversight."

The Solution

As a software engineer, I started thinking: Can I automate this? Can I make a system that does the heavy lifting so I can focus on being creative?

The Grainbound is the result of that question. It’s a repository of frameworks designed to bypass the "blank page" paralysis.
